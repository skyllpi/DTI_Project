Common Issues

Error: "Could not establish connection. Receiving end does not exist."

This error occurs when the extension tries to communicate with a content script that hasn't been loaded yet.

Solution: Make sure you're on a YouTube page before trying to use the extension. If you set up your API key while not on YouTube, reload any YouTube tabs you have open.

No summary appears when hovering

This could be due to several reasons:

  • Your API key is not valid or not set up
  • There's a CORS (Cross-Origin Resource Sharing) issue
  • The YouTube page structure has changed
  • The extension didn't initialize properly

Solution: Click the extension icon, verify your API key is correct, save settings, and refresh the YouTube page.

API key errors

If you're seeing API key errors:

  • Make sure you've entered the correct API key from Google Gemini
  • Check if your API key has the necessary permissions
  • Verify your API key hasn't expired or reached its quota limit

API model errors

This extension uses the Gemini 2.0 Flash model. If you see errors about the model not being found:

  • Make sure your API key has access to the "gemini-2.0-flash" model
  • Verify that you're using the correct API version (currently v1beta)
  • Check that your API key has appropriate permissions for generateContent
  • Consider checking the Gemini documentation for updates about model availability
